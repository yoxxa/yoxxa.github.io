Moving to a terminal-based workflow

  • Mar 8, 2024
  • One minute read

Call it a bandwagon but I have moved to a terminal workflow for efficiency sake. Upside is, it makes me feel like a power user, which is half of the battle.

My current setup includes:

  • Zsh
  • Tmux
  • Alacritty
  • Neovim

Whilst these applications are all vastly configurable, I’ve elected to keep them as minimal, readable and understandable as possible for my own sake.

Config files can be found in this repository.
