Call it a bandwagon but I have moved to a terminal workflow for efficiency sake. Upside is, it makes me feel like a power user, which is half of the battle. Downside is, I'm now doomed to gloat about the tools I use.

My new setup includes:
- Bash -> Zsh
- Tmux
- Xfce Terminal -> Alacritty
- VSCodium -> Neovim
- LazyGit

Whilst these applications are all vastly configurable, I've elected to keep them as minimal, readable and understandable as possible for my own sake.

[Config files can be found in this repository.](https://github.com/yoxxa/dotfiles)
